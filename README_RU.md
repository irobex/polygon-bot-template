## polygon-bot-template

Шаблон бота для Polygon: мониторинг блоков/событий, готов к `fastpolygon.tech` RPC.

### Быстрый старт

```bash
npm i ethers
```

```bash
export FP_API_KEY="ВАШ_КЛЮЧ"
node bot.js
```

### Endpoint
- `https://rpc.fastpolygon.tech/v1/{API_KEY}`

### Файлы
- `bot.js` — минимальный мониторинг новых блоков
